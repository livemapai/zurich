AI-Generated Textures from Satellite Imagery
=============================================

This directory contains textures generated by texture_generator.py using
Gemini 2.5 Flash Image, informed by actual Zurich satellite imagery.

TEXTURE TYPES
-------------

Building Walls:
- residential_plaster/  - Warm cream/beige Swiss stucco for residential
- commercial_facade/    - Modern glass and metal for commercial buildings
- industrial_concrete/  - Weathered concrete for industrial structures
- historic_plaster/     - Aged plaster for Altstadt historic buildings

Roofing:
- roof_terracotta/      - Traditional Swiss terracotta clay tiles
- roof_slate/           - Dark slate tiles for public/historic buildings
- roof_flat/            - Modern flat roof membrane/gravel

Ground:
- sidewalk_concrete/    - European concrete pavers for sidewalks

GENERATING TEXTURES
-------------------

To generate all textures from a reference satellite tile:

    python -c "
    from scripts.tile_pipeline.texture_generator import generate_all_textures_from_reference
    from scripts.tile_pipeline.sources.satellite import fetch_satellite_tile
    sat = fetch_satellite_tile(16, 34322, 22950)
    generate_all_textures_from_reference(sat)
    "

Or generate a single texture type:

    python -c "
    from scripts.tile_pipeline.texture_generator import generate_texture_from_satellite
    from scripts.tile_pipeline.sources.satellite import fetch_satellite_tile
    sat = fetch_satellite_tile(16, 34322, 22950)
    generate_texture_from_satellite('residential_plaster', satellite_tile=sat)
    "

FILE STRUCTURE
--------------

Each texture type directory contains:
- albedo.png  - Base color texture (1024x1024, seamless/tileable)
- .gitkeep    - Ensures directory is tracked

USAGE IN RENDERS
----------------

Enable textures in style_presets.py:

    DataDrivenStylePreset(
        ...
        use_textures=True,
        texture_scale=0.1,  # meters per texture repeat
        texture_from_satellite=True,
    )

Or use a pre-configured textured preset:

    python -m scripts.tile_pipeline.cli render --preset textured

REQUIREMENTS
------------

- GOOGLE_API_KEY environment variable set
- Gemini 2.5 Flash Image API access (free tier: 1500 requests/day)
